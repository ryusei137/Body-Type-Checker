# Body Type Checker

## 概要

Pythonで作成した体型判定プログラムです。

ユーザーが体脂肪率と筋肉率を入力すると、入力された数値をもとに体型を判定し、結果を表示します。

## 作成目的

体脂肪率と筋肉率という複数のデータを組み合わせて判定を行うことで、条件分岐や論理演算子の理解を深めることを目的として作成しました。

また、自身の筋力トレーニング経験を活かし、健康管理に関連するプログラムを作成しました。

## 使用技術

* Python
* input
* float型
* if / elif / else
* 論理演算子（and）
* 比較演算子
* f-string

## 機能

* 体脂肪率の入力
* 筋肉率の入力
* 体型判定
* 判定結果の表示

## 判定例

* 痩せ型
* 細身筋肉質
* 筋肉質
* 運動不足
* 標準
* 隠れ肥満
* 肥満型
* 固太り

## 工夫した点

体脂肪率と筋肉率の2つの数値を組み合わせて判定を行うようにしました。

複数の条件を設定することで、単純な数値判定ではなく、さまざまな体型パターンを表現できるよう工夫しました。

また、if / elif / else を活用して条件分岐を整理し、判定結果を分かりやすく表示できるようにしました。

## 実行方法

python bodyfat.py
